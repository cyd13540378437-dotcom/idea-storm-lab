# Idea Storm Lab

Idea Storm Lab 是一个面向早期创业团队的邀请制脑暴工具。

它用于收集团队成员的创业想法，并通过 LLM 将模糊想法整理成可讨论的结构化分析，包括内容提取、竞品线索、核心用户群体、业务流程、运营模式、资本视角业务故事、产品核心能力、技术选型、风险建议，以及标准商业模式画布。

项目当前重点是帮助远程协作的小团队完成三个动作：

- 记录想法：每个成员可以提交自己的文字、图片和文档材料。
- 分析想法：系统根据项目内的分析 Skill 生成专业分析。
- 共享想法：所有成员可以查看彼此提交的想法，但只能编辑自己的内容。

## Version History

### v0.3.0

- 接入项目内分析 Skill：`skills/analysis_skill.md`。
- 将 LLM 分析从裸提示词升级为可迭代的创业分析专家规则。
- 新增 Skill 优化手册和服务器部署手册。
- 支持 Qwen / DashScope OpenAI 兼容接口。

### v0.2.0

- 调整页面信息架构为：写想法、分析结果、灵感墙。
- 提交想法后自动进入分析结果页面。
- 将灵感墙调整为独立菜单，作为团队想法总览。
- 分析结果页面优先展示核心分析内容，原始想法和附件后置。

### v0.1.0

- 实现邀请制注册和登录。
- 支持不限人数成员身份。
- 支持每个成员创建多个独立想法实例。
- 支持想法列表展示作者、更新时间和附件数量。
- 支持作者编辑自己的想法，其他成员只读。
- 支持图片和文档附件上传。
- 支持自动生成文字分析和商业模式画布。
- 使用本地 SQLite 保存数据，本地 uploads 目录保存附件。
